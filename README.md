# Redux CRUD Dashboard

## Tech Stack

- React
- Redux Toolkit
- Axios

## Features

- Fetch data from public API
- CRUD operations using Redux
- Dashboard table view
- Loading & error handling

- Edit user functionality with modal
- Tailwind CSS based UI

## Steps to Run

1. git clone <repo-url>
2. cd redux-crud-dashboard
3. npm install
4. npm run dev

## Note

CRUD operations are handled locally using Redux Toolkit.
